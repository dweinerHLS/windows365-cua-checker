Check your tenant readiness for CUA

Sign in with your Microsoft Global Administrator account. This tool will automatically validate all technical prerequisites required to run Cloud PC pools for Computer Use Agent (CUA) workloads in Microsoft Copilot Studio.

What gets checked automatically

Windows 365 & Azure Virtual Desktop service principals
Microsoft Remote Desktop RDP authentication configuration
Consent prompt suppression (targetDeviceGroups)
Intune device enrollment restrictions
Entra ID P1 / admin role assignment
Power Platform billing, environments & Cloud PC feature
Manual checklist for remaining items
ℹ️ On first use, Microsoft will ask you to approve permissions on behalf of your organization. This is a standard one-time consent for your tenant — no configuration needed.

🔒 All checks run in your browser. No data is sent to any external server.
